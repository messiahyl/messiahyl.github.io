Core Features
为开发者设计的
三个核心能力
不是又一个聊天机器人。TechDoc Agent 是专门为技术文档理解而构建的垂直 Agent。
📄

架构自动分析

上传任意技术文档，Agent 自动识别核心架构模式、组件依赖关系和数据流向，输出结构化分析报告。

💬

智能文档问答

基于文档上下文的多轮对话。精准回答关于技术选型、设计决策和实现细节的问题，拒绝幻觉。

🔌

MCP 工具集成

通过 Model Context Protocol 即插即用连接文件系统、搜索引擎、数据库等外部工具，扩展 Agent 能力边界。

API Reference
简洁的 REST API
三个端点覆盖所有场景，5 分钟集成到你的系统。
POST /api/v1/agent/analyze 文档分析 
// 请求
{
  "content": "你的技术文档内容..."
}

// 响应 — 自动输出架构分析、关键决策、风险评估
{
  "reply": "## 架构概览\n- 核心模式: 微服务 + 事件驱动\n- 关键决策: 选用 Kafka 作为消息总线..."
}
POST /api/v1/agent/chat 通用对话 
// 支持多轮对话
{
  "message": "解释一下微服务架构的核心模式",
  "history": [
    { "role": "user", "content": "上一步的问题..." },
    { "role": "assistant", "content": "上一步的回答..." }
  ]
}
POST /api/v1/agent/ask 文档问答 
// 基于指定文档上下文精准回答
{
  "question": "这个系统用了什么消息队列？",
  "documentContext": "文档上下文...",
  "history": []
}
Quick Start
30 秒启动你的 Agent
01

配置 API Key

设置 LLM_API_KEY 环境变量，支持 DeepSeek / Qwen / Claude 一键切换。

02

启动服务

运行 mvn spring-boot:run，MCP Server 自动连接。

03

调用 API

POST 请求 /api/v1/agent/analyze，获取结构化分析结果。
